PQC Geometry & Topology Analysis

This project provides tools for analyzing the geometric and topological properties of Parameterized Quantum Circuits (PQCs) using PennyLane, PyTorch, and Ripser.

Installation

Install dependencies with:

pip install pennylane pennylane-lightning[torch] torch numpy tqdm tabulate ripser

Quick Usage

Import and call the analysis functions with your PQC.

from pqc_analysis.geometry_analysis import pqc_geometry_analysis
from pqc_analysis.topology_analysis import pqc_topology_analysis

Geometry Analysis

pqc_geometry_analysis evaluates the expressibility of PQCs using the Fubini-Study metric tensor.

summary = pqc_geometry_analysis(
    pqc=my_pqc,
    n_params=4,
    n_qubits=2,
    n_samples=500,
    init_strategy='uniform'
)

Function Signature:

pqc_geometry_analysis(pqc, n_params, n_qubits, n_samples=100, init_strategy='uniform', init_thetas=None, **kwargs)

Returns:

  • Total Volume: Reachable state space volume
  • Avg Local Volume: Determinant of the metric tensor
  • Avg Metric Rank: Rank of the metric matrix
  • Redundant Param Ratio: % of rank-deficient parameterizations
  • Avg Gamma: Condition number (inverse eigenvalue spread)

Topology Analysis

pqc_topology_analysis estimates persistent homology from sampled PQC outputs.

entropy_values = pqc_topology_analysis(
    pqc=my_pqc_topology,
    n_params=2,
    n_qubits=2,
    n_samples=200,
    max_dim=1
)

Function Signature:

pqc_topology_analysis(pqc, n_params, n_qubits, n_samples=100, max_dim=2, init_strategy='normal', init_thetas=None, **kwargs)
