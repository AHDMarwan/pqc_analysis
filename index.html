PQC Analysis

This repository provides Python code for analyzing the geometric and topological properties of Parameterized Quantum Circuits (PQCs) using PennyLane, PyTorch, and Ripser.

Installation

Install the required libraries using pip:

pip install pennylane pennylane-lightning[torch] torch numpy tqdm tabulate ripser

Package Structure

  • geometry_analysis.py: Compute Fubini-Study metric tensor and related metrics.
  • topology_analysis.py: Perform topological data analysis using persistent homology.

Usage

Geometry Analysis

pqc_geometry_analysis analyzes Fubini-Study metric tensor of a PQC.

from pqc_analysis.geometry_analysis import pqc_geometry_analysis

def my_pqc(params):
    qml.RX(params[0], wires=0)
    qml.RZ(params[1], wires=0)
    qml.RX(params[2], wires=1)
    qml.RZ(params[3], wires=1)
    return qml.state()

summary = pqc_geometry_analysis(
    pqc=my_pqc,
    n_params=4,
    n_qubits=2,
    n_samples=500,
    init_strategy='uniform'
)
print(summary)

Topology Analysis

pqc_topology_analysis infers topological features using persistent homology.

from pqc_analysis.topology_analysis import pqc_topology_analysis

def my_pqc_topology(params):
    qml.RX(params[0], wires=0)
    qml.RZ(params[1], wires=0)
    qml.CNOT(wires=[0, 1])
    return qml.state()

entropy_values = pqc_topology_analysis(
    pqc=my_pqc_topology,
    n_params=2,
    n_qubits=2,
    n_samples=200,
    max_dim=1
)
print(entropy_values)
